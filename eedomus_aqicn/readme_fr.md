Ce plugin permet d'obtenir différents indices de qualité de l'air via l'API du service AQICN. 


<br>

# Informations nécessaires

Vous devez saisir les informations suivantes :

- Token AQICN: Vous pouvez obtenir ce token ici : https://aqicn.org/data-platform/token/#/

- Latitude du site dont vous souhaitez obtenir les indices de qualité de l'air.

- Longitude du site dont vous souhaitez obtenir les indices de qualité de l'air.

En renseignant votre longitude et votre latitude, vous obtiendrez les données de qualité de l'air issues de la station de mesure la plus proche.


<br>

# Périphériques créés sur votre box eedomus après l'installation du plugin

Après installation du plugin, vous obtiendrez sur votre box eedomus les périphériques suivants :

- Station de mesure IQA

- Indice général de qualité de l'air

- Polluant prédominant

- Humidité

- NO2

- O3

- Pression

- PM 10

- PM 2.5

- Température

- Vent

- Rafale de vent

<br>

# Remarques

Vous pouvez masquer ou désactiver les périphériques que vous le souhaitez pas afficher.

Vous pouvez ajuster de polling de chaque périphérique (fixé à 15 minutes par défaut).

Certaines stations de mesure ne permettent pas d'obtenir tous les indices de qualité de l'air. Dans ce cas, vous aurez un message d'erreur vous indiquant qu'il est impossible d'obtenir la valeur correspondante. Il vous suffit alors de désactiver le périphérique concerné.

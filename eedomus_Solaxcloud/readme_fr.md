Plugin permettant d'intégrer sur la box eedomus les onduleurs solaires photovoltaïques de la marque Solax via Solar Cloud. Il ne s'agit donc pas d'une intégration en local, mais via le cloud de Solax.

<br>

# Comment installer le plugin Solax Cloud sur votre box eedomus ?

- Sur le portail eedomus (secure.eedomus.com), saisissez votre identifiant et votre mot-de-passe puis rendez-vous à Configuration / Ajouter ou supprimer un périphérique / Store eedomus / Solax Cloud et cliquez sur "Créer".

- Dans la fenêtre qui s'ouvre après l'installation du plugin, associez le nouveau périphérique à l'une des pièces disponibles, puis saisissez le "Registration No" de votre onduleur Solax (indiqué sur celui-ci), ainsi que le "token Solax Cloud API" (obtenu au préalable via votre compte Solax). Enfin, cliquez sur "Créer".

<br>

# Périphériques créés sur la box eedomus après l'installation du plugin

Suite à l'installation du plugin Solax Cloud sur votre box eedomus, vous aurez 13 nouveaux périphériques qui vont vous permettre d'obtenir la quasi-totalité des informations relatives à votre onduleur Solax fournies par Solax Cloud API.

Si l'un de ces périphériques/canaux ne vous est pas utile (si vous n'avez pas de batterie, par exemple), vous pouvez le désactiver depuis l'écran de configuration du périphérique (dans Paramètres Expert). Il vaut mieux ne pas le supprimer car il est associé aux autres périphériques et il vous faudrait le dissocier avant de l'éliminer pour ne pas supprimer tous les périphériques/canaux.

Le polling de chacun de ces 13 périphériques est fixé à 1 minute. Par conséquent, le rafraîchissement s'effectue toutes les minutes.

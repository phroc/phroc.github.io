
## Pushover, qu’est-ce que c’est?

Pushover es un service de notification basé sur des applications mobiles pour iOS et Android.

Ces applications sont payantes mais il est possible de les tester gratuitement pendant un certain temps.


## À quoi sert le périphérique Pushover pour eedomus?

Ce périphérique permet d’intégrer simplement les notifications Pushover sur la plate-forme eedomus.


## Pré-requis

Pour intégrer les notifications Pushover sur la plate-forme eedomus il faut:

- créer un compte sur Pushover (https://pushover.net)
- installer l’application Pushover sur son téléphone/tablette iOS ou Android
- noter la “User Key” fournie par Pushover lors de la création du compte
- créer une “application” sur Pushover (ex: “Notifications eedomus”) et noter l’“API Token” fournie par Pushover à ce moment-là.


## Informations à renseigner lors de la création du périphérique Pushover sur eedomus

- User Key Pushover

- Api Token Pushover

- Message: Texte de la notification

- Sound: Son de la notification (option à choisir dans un menu déroulant)

- Priority: Niveau de priorité de la notification (option à choisir dans un menu déroulant).


## ¿Qué es Pushover?

Pushover es un servicio de notificación basado en aplicaciones móviles para iOS y Android.

Estas aplicaciones son de pago pero puede probarlas gratuitamente durante un tiempo.


## ¿Para qué sirve el periférico Pushover para eedomus?

Este periférico permite integrar de forma sencilla las notificaciones Pushover en la plataforma  eedomus.


## Requisitos previos

Para integrar las notificaciones Pushover en la plataforma eedomus, debe:

- crear una cuenta en Pushover (https://pushover.net)
- instalar la aplicación Pushover en su  smartphone/tablet iOS o Android.
- apuntar la “User Key” que le proporciona Pushover al crear la cuenta.
- crear una “aplicación” en Pushover (por ejemplo: “Notificaciones eedomus”) y apuntar el “API Token” que le proporciona Pushover al hacerlo.


## Datos necesarios para crear el periférico Pushover en eedomus

- User Key Pushover

- Api Token Pushover

- Message: Texto de la notificación

- Sound: Sonido de la notificación (debe elegir una de la opciones disponibles en el menú desplegable)

- Priority: Nivel de prioridad de la notificación (debe elegir una de la opciones disponibles en el menú desplegable)
